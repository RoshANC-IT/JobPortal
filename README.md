# JobPortal

Job Portal with Full Authentication (MERN Stack)
This project is a job portal application built using the MERN stack (MongoDB, Express, React, Node.js) with shadcn ui  that includes full authentication features.
1:User Features:  ✅
--- Users can create accounts, apply for jobs, and track the status of their applications.
2:Admin Features:  ✅
--- Admins can create and manage company profiles, job listings, and accept or reject employee applications.
3:Auto Recommendation System: 
--- Currently in progress, the system aims to automatically recommend jobs to users based on their profiles and preferences.Means user are eligible or not
